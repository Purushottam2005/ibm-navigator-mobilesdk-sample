IBMECMAuthenticatorDataSource

public protocol IBMECMAuthenticatorDataSource

The data source that allows you to inject a custom third-party authentication mechanism into this framework.

  • customAuthenticationIsRequiredForUrl(_:)

    This method is called every time authentication is about to take place. When this method is invoked, it gives you the opportunity to inspect the URL and decide whether the custom authentication mechanism is applicable. This decision might be as simple as the URL pointing to a well-known host name or it might require a round trip to that server in order to understand what authentication method is required. Either a true or false value must be returned.

    Declaration

    Swift

    func customAuthenticationIsRequiredForUrl(url: NSURL) -> Bool

    Parameters

    url

    the URL to log in to

    Return Value

    true if you intend to provide custom authentication for this URL, false otherwise

  • customAuthenticationForUrl(_:)

    This method is called only if the method ‘customAuthenticationIsRequiredForUrl’ returned a value of true for the same URL. If so, the IBMECMCore framework invokes this method to solicit the custom authentication method that you want to use. The return of this method must be an instance of IBMECMAuthenticator so that IBMECMCore has a reference to the class and calls it as it needs it.

    Declaration

    Swift

    func customAuthenticationForUrl(url: NSURL) -> IBMECMAuthenticator

    Parameters

    url

    the URL to log in to

    Return Value

    an instance of IBMECMAuthenticator that implements the custom authentication logic
