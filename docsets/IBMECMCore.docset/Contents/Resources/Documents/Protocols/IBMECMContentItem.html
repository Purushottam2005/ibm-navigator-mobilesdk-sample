IBMECMContentItem

public protocol IBMECMContentItem : IBMECMBaseObject

Represents a single content item in the repository. Content items include documents (has content), folders (for storage), and other items stored in a repository. Each content item is associated with a content class that describes the metadata for the item.

If the content item is a document, it can be versioned and grouped together by its version series id.
