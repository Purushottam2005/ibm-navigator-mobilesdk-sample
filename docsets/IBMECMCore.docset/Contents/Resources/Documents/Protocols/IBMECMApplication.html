IBMECMApplication

public protocol IBMECMApplication: class

Represents the main application or device interfacing with this SDK.

An application object is the top-level object of this SDK. It holds all the IBMECMDesktop objects that this application will be accessing. A default desktop is usually set in the IBM Content Navigator server and is provided to this SDK by the baseURL set in IBMECMFactory.sharedInstance.getApplication(String). You can also provide or switch to a different desktop by providing a desktopId param in the baseURL.

Typically you will want to call loadDesktop() to load each desktop information. By loading the desktop, you will bring down all the settings, information, and repositories this desktop can access. You can also call login() to authenticate to the default repository for the default desktop.

There should only be one application instance that represents your application. You can obtain this singleton instance from IBMECMFactory.sharedInstance.getApplication(String)
