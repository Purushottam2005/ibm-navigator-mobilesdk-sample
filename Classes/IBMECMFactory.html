IBMECMFactory

public class IBMECMFactory: NSObject

A singleton factory that provides entry points to the top-level API objects, such as IBMECMApplication or IBMECMDesktop.

Most of the time you will start with the following: 

IBMECMApplication application = IBMECMFactory.sharedInstance.getApplication(baseURL)
